# Security Policy

## Supported Versions

Currently, only the `main` branch is actively supported for security updates.

## Reporting a Vulnerability

Security is a core feature of Guptrix. If you discover a security vulnerability, we kindly ask you to report it privately rather than opening a public issue. 

Please send a detailed report to the security team at `security@guptrix.example.com`. 

Your report should include:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact and an example attack scenario.

We will acknowledge receipt of your vulnerability report within 48 hours and strive to send you regular updates about our progress. If a fix is implemented, we will coordinate with you to publish a responsible disclosure.
